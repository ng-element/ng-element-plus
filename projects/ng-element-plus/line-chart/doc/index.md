### Attribute

| 参数       | 说明           | 类型    | 可选值 | 默认值 |
| ---------- | -------------- | ------- | ------ | ------ |
| option     | echarts配置项  | any     | —      | —      |
| width      | 宽度           | string  | —      | 100%   |
| height     | 高度           | string  | —      | 300px  |
| autoResize | 是否自适应尺寸 | boolean | —      | true   |

### Events

| 参数    | 说明            | 回调参数    |
| ------- | --------------- | ----------- |
| onReady | echarts实例创建 | echarts实例 |
